## 🔌 API Endpoints

### Аутентификация
```
POST   /api/auth/register          Регистрация
POST   /api/auth/login             Вход
GET    /api/auth/me                Текущий пользователь
POST   /api/auth/refresh           Обновить токен
```

### Сканирование
```
POST   /api/document-inspector/detect     Сканировать документ
GET    /api/document-inspector/health     Проверка модели
```

### История сканов
```
GET    /api/scans/                 Мои сканы
GET    /api/scans/stats            Моя статистика
GET    /api/scans/{id}             Детали скана
GET    /api/scans/{id}/download    Скачать PDF
DELETE /api/scans/{id}             Удалить скан
```

### Админ (требуется роль admin)
```
GET    /api/admin/users            Все пользователи
GET    /api/admin/users/{id}/scans Сканы пользователя
GET    /api/admin/stats            Системная статистика
DELETE /api/admin/users/{id}       Удалить пользователя
```
